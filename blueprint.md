# Blueprint: Boracay Reservation Website

## Overview

This project is a reservation website for tours and activities in Boracay. It allows users to browse available activities, view details, and make reservations. The project also includes an admin panel for managing reservations.

## Project Structure

*   `index.html`: The main landing page of the website.
*   `style.css`: The main stylesheet for the website.
*   `main.js`: The main JavaScript file for the website.
*   `admin.html`: The admin panel for managing reservations.

## Current Plan

### Create an Admin Site

*   **Objective:** Build a password-protected admin page to manage reservations.
*   **Steps:**
    1.  **Create `admin.html`:** A new HTML file for the admin interface.
    2.  **Design Admin Page:** Add a table to display a list of reservations.
    3.  **Create `blueprint.md`:** Document the project and the new admin page.
    4.  **Implement Login:** Add a simple login system to restrict access.
    5.  **Fetch and Display Data:** Load and show reservation data on the admin page.
    6.  **Add Management Features:** Include functionality to approve or delete reservations.
